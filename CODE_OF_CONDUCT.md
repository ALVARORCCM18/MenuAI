# MenuIA Code of Conduct

This project welcomes collaboration, review, and feedback from people with different backgrounds and experience levels.

We expect everyone participating in the project to act with professionalism, respect, and care. This applies to issues, pull requests, code reviews, documentation edits, and any other project communication.

## Expected behavior

- Be respectful and constructive.
- Be patient with questions, mistakes, and differing viewpoints.
- Focus criticism on the code or proposal, not the person.
- Avoid harassment, exclusionary language, or offensive behavior.
- Respect privacy and do not share personal or sensitive information.

## Unacceptable behavior

- Harassment, intimidation, or threats.
- Discriminatory, sexual, or violent language.
- Doxing or sharing private information without consent.
- Repeated disruptive behavior after being asked to stop.

## Reporting issues

If you experience or witness behavior that violates this code of conduct, contact the project maintainer privately at [alvaro.rccm18@gmail.com](mailto:alvaro.rccm18@gmail.com).

Please include:

- What happened.
- Where it happened.
- Any links, screenshots, or context that help describe the situation.

Reports will be handled privately and as promptly as possible.
